# Logout

Developers can enable user logout (manager removal) through Portkey SDK

## Core SDK

### Usage

```TypeScript
import { did } from '@portkey/did-ui-react';

did.logout({ chainId: 'CHAIN_ID' }); // chainId required
```

### TypeScript

```TypeScript
logout(params: EditManagerParams): Promise<boolean>;
```
